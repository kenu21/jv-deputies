Deputies of Verkhovna Rada of Ukraine Deputy Statistics API

Available requests:

/deputy - view all deputies, select page: /deputy?page=INSERT_PAGE_NUMBER, 10 deputies per page

/deputy/byConvocation/INSERT_CONVOCATION_NUMBER - view all deputies by convocation number

/deputy/notPoliticians - view all deputies whose major proficiency is not politician

/deputy/awarded - view deputies possessing various awards

/deputy/getGroupedByParty - view deputies sorted by party affiliation

/deputy/notUkrainian - view deputies whose nationality is not Ukrainian

/deputy/byNationality/INSERT_NATIONALITY - view deputies by nationality, possible requests "українець", "укр" etc.

/deputy/partiesByConvocation/INSERT_CONVOCATION_NUMBER - view political parties within single parliament convocation

/deputy/leastActiveTerms - view deputies with shortest terms of parliament activity, select page: /deputy/leastActiveTerms?page=INSERT_PAGE_NUMBER, 10 deputies per page

/deputy/byUniversity/INSERT_UNIVERSITY_NAME - view deputies graduated from particular university, possible requests: "КПІ", "НАУ", etc.

/deputy/partySwitchers - view deputies who changed their party affiliation most frequently
